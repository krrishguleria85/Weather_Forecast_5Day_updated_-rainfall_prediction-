# AI Weather Forecasting System

Hybrid ML + Weather API forecasting application.

##Website URL:
https://weather-ai-forecast.onrender.com


## Features
- 5-day weather forecast
- Rain prediction model
- Dynamic weather UI
- City geolocation
- Hybrid ML + rule-based weather engine

## Tech Stack
Python, Flask, Scikit-Learn, Pandas, OpenWeather API

## Model Performance
Accuracy: 85.21%
